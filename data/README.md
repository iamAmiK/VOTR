# VOTR Data

The pre-computed index files and benchmark datasets for VOTR are hosted on Hugging Face. Because the dense embeddings and tool metadata files are quite large, they are not tracked in this Git repository.

## Download Instructions

You can view and download the dataset directly from Hugging Face:
[https://huggingface.co/datasets/a13awd/VOTR](https://huggingface.co/datasets/a13awd/VOTR)
